# 🌐 Speed - LoTest

**Speed - LoTest** est une application web simple permettant de tester la qualité d'une connexion Internet.  
Elle mesure les éléments suivants :
- 📶 **Vitesse de téléchargement (Mbps)**
- 🛰️ **Ping (ms)**
- 🔁 **Jitter (ms)**

## 🚀 Technologies utilisées

- HTML5
- CSS3
- JavaScript

## 🔧 Fonctionnalités

- Test réel de téléchargement via un fichier distant
- Simulation réaliste du ping et du jitter
- Interface moderne et responsive
- Barre de progression dynamique et animation des résultats
